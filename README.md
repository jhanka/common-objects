# common-objects
Common objects - Public jar repository

Usage:

<repositories>
    <repository>
        <id>common-objects-mvn-repo</id>
        <url>https://raw.github.com/jhanka/common-objects/mvn-repo/</url>
        <snapshots>
            <enabled>true</enabled>
            <updatePolicy>always</updatePolicy>
        </snapshots>
    </repository>
</repositories>
